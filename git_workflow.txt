# Git Workflow for Task App Project

1. Initialize repository
   git init

2. Add remote repository
   git remote add origin https://github.com/username/task-app.git

3. Add all files
   git add .

4. Commit changes
   git commit -m "Implemented email validation, welcome email, and user list"

5. Push changes to remote
   git branch -M main
   git push -u origin main
